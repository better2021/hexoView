this的指向

在ES5中，this的指向，始终坚持一个原则：this永远指向最后调用它的那个对象

fn() 最后仍然是被 window 调用的。所以 this 指向的也就是 window

箭头函数

在ES6中箭头函数是可以避免ES5中this的坑，箭头函数的this始终指向函数定义的this，而非执行
